1. pengenalan html

HTML, jika anda mendengar apa sih HTML. Mungkin untuk sebagian orang HTML sangatlah asing, apalagi kalau anda bukan bagian dari anak RPL maupun TKJ yang berhubungan dengan IT.

Tapi kalian harus tahu terlebih dahulu, apa itu bahasa pemrograman? Bahasa pemrograman adalah serangkaian kata yang diwujudkan dalam sebuah instruksi ataupun perintah yang akan menjadi deretan atau baris suatu konten.

Disini, saya akan menjelaskan secara singkat dan padat. Apa itu HTML?

HTML biasa disebut juga dengan Hypertext Markup Language. HTML termasuk kedalam bahasa pemrograman basic yang dapat menterjemahkan bahasa Anda kedalam bahasa komputer dengan kalimat perintah atau tag perintah didalam HTML.

HTML juga termasuk sebagai penyusun awal dari struktur pembuatan website. Dan, dapat terbantu oleh CSS untuk memperindah dan membuat tampilan menjadi menarik.

Jika anda menginginkan, menjadi seorang developer anda wajib mempelajari HTML tersebut.

1.1 cara membuat text (h1-h6, p, a)

Sebelum itu, perkenalkan dulu apa itu yang dinamakan "Heading", "Paragraf" dan "a atau bisa disebut dengan link."

Heading bisa dikatakan pula dengan 'h1-h6'. Semakin besar angkanya semakin kecil tulisan yang dihasilkan. Jika kalian ingin membuat sebuah judul, maka gunakan 'h1' tetapi jika kalian ingin membuat tulisan lebih kecil maka gunakan 'h2' hingga 'h6'.

Sedangkan, paragraf bisa dikatakan dengan 'p' ukuran standar dari sebuah paragraf. Tag paragraf biasanya kita gunakan seperti halnya membuat kalimat yang bersifat sebagai isi.

Kemudian, perintah 'a' digunakan untuk peng-link an. Tag 'a' dapat digunakan untuk membuka tab baru atau memindahkan halaman satu ke halaman yang lain. Caranya : Tulis tag 'a' kemudian kalian 'enter', dan akan muncul tampilan seperti , 'a href=""''/a'. Untuk halaman peng-linkan diisi didalam href, dan untuk isi penulisan 'a href=""'link'/a'. Namun, di html ditulis setelah kurung tutup dari perintah p 'a href="adhityaap.github.io"'link'/a'.

1. tutorial css

1.1 css didalam html

Sebelum anda mengenal lebih jauh, saya akan menjelaskan perbedaan CSS didalam HTML, Inline, serta Di luar HTML.

Tapi, apasih CSS? CSS adalah Cascading Style Sheets. CSS terbagi menjadi 3 golong, yaitu : internal, inline, dan eksternal.

Pertama, CSS didalam HTML adalah CSS yang termasuk kedalam CSS internal. Kenapa? Karena letak perintah tag di tulis didalam 'head' dan dibawah 'title' serta menggunakan perintah tag 'style'.

Yang kedua, CSS inline adalah CSS yang ditulis di atribut elemen HTML. Tetapi, inline biasanya hanya digunakan untuk menambakan suatu style pada elemen.

Dan yang terakhir, CSS Eksternal adalah sebuah CSS yang ditulis terpisah pada HTML. Bisanya untung menggabungkan CSS Eksternal dengan HTML anda memerlukan sebuah tag perintah 'link'.

Nah sekarang, udah tau kan perbedaan ketiga CSS dan apa aja kegunaannya.

Yuk, kalian simak cara-cara yang akan saya jelaskan dibawah ini.

1.2 css diluar html

CSS diluar HTML, biasa disebut juga dengan CSS Eksternal. Manfaat CSS Eksternal adalah ukuran HTML menjadi lebih kecil dan akan menjadi struktur yang rapih.

Dalam sebuah HTML, CSS Eksternal terletak pada bagian 'head' di halaman HTML.

Langkah pertama membuat CSS Eksternal adalah anda harus membuat File CSS didalam folder yang sama dengan File HTML.

Setelah anda membuat file, pastikan anda sudah save as file dengan ekstensi CSS. Jika sudah, tulis tag perintah dibagian bawah 'head' kemudian, anda bisa memanggil folder CSS anda dengan tag 'link' dan tambah dengan atribut "href". Pada atribut "href" anda bisa ubah sesuai dengan nama file CSS anda.

Setelah terhubung, maka anda sudah bisa mengakses Style dihalaman HTML anda.

1.6 cara menambahkan background

Pastinya anda menginginkan website atau konten Anda terlihat menarik bukan? Mau tau bagaimana cara menambahkan background di html? Coba anda simak cara berikut ini !

Dalam sebuah website memerlukan yang namanya keunikan dan kemenarikan suatu konten. Dengan anda menambahkan background pada konten Anda, maka semua orang akan tertarik melihatnya.

Ada dua cara menambahkan background, yaitu : Pertama, jika anda menginginkan background solid. Maka, anda hanya perlu menggunakan perintah seperti ini :
.kontenfoto{
Background - color : ◻White ;
}

Kedua, jika anda ingin background anda tampak foto. Maka, anda hanya perlu menggunakan perintah seperti ini :
.kontenfoto{
Background - Image : url (nama file foto) ;
Background - size : Cover ;
Background - Repeat : no - repeat;
}

Simple dan mudah bukan untuk anda praktekan. Bahkan, website anda akan terlihat menarik jika anda mempraktekkan ini.

1.7 cara mengatur gambar

Tata letak gambar sangat penting seperti tata letak tulisan. Kenapa? Karena, semua padangan orang pertama kali yang dilihat adalah gambar dan tulisan.

Tapi gambar anda masih berantakan? Hmm ... Jangan khawatir saya ada solusi cepat dan praktis, bahkan gambar ada akan terlihat proporsi.

Ketika anda ingin mengatur gambar, saya merekomendasikan untuk anda mengatur salah satu bagian saja. Baik dari width ataupun height.

Sebab, jika anda mengatur kedua-duanya foto anda akan terlihat seperti gepeng. Maka, saya hanya merekomendasikan untuk pilih salah satu.

Seperti penjelasan sebelumnya, mengatur image didalam Div Class harus ditulis secara spesifik nama class dan perintahnya.

Berikut contohnya :
.kontenfoto img{
Width: 100px;
}

Mudah bukan caranya, jangan lupa anda praktekan jika anda menginginkan hasil yang memuaskan.

1.8 cara mengatur posisi item

Ada kendala diukuran background dan lainnya? Pasti ada hal yang anda lupa! Apakah anda sudah menambahkan margin dan padding kedalam konten Anda?

Belum... Sepertinya anda harus melihat ini dan mempraktikkan kedalam konten Anda, agar konten ada semakin sempurna.

Sebelumnya, anda harus mengenal apa itu margin dan padding. Okee, pertama untuk margin, margin adalah perintah untuk mendorong dari titik terpinggir konten untuk ke luar. Sedangkan, padding adalah mendorong dari titik terluar konten untuk kedalam.

Berikut adalah contoh mengatur margin dan padding :

.kontenfoto{
Margin : 10 px
Padding : 20 px
}

Simple tetapi anda akan melihat perubahan pada konten Anda, konten Anda akan menarik dan pastinya sangat membantu jarak.
